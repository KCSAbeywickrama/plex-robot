# Robot Plex

**Robot Plex** became the runner-up robot in the robot design competition which was organized as a requisite of EN2532 - Robot Design and Competition module of the Electronics and Telecommunication Engineering department of the University of Moratuwa. This repo contains all project files of our robot.
